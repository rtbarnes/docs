title: 'Tests de référence OpenAPI'
description: 'Ensemble de tests pour différents styles de référence à une spec OpenAPI'
---

Cette section teste les références à des specs OpenAPI :

- **Relative** – `openapi/openapi-example.json` (chemin relatif à la racine du projet)
- **Absolue** – `/openapi/openapi-example.json` (absolu depuis la racine du dépôt)
- **Même répertoire** – `docs/api-reference/openapi-test/local-spec.json` (spec colocalisée avec le MDX)
- **Distante** – `https://petstore3.swagger.io/api/v3/openapi.json` (URL hébergée)
