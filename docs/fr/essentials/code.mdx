## Code en ligne

Pour indiquer qu'un `mot` ou une `expression` est du code, encadrez-le de backticks (`).

```
Pour indiquer qu'un `mot` ou une `expression` est du code, encadrez-le de backticks (`).
```

## Blocs de code

Utilisez les [blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en encadrant le code de trois backticks, et faites suivre les backticks d'ouverture du langage de programmation de votre extrait pour obtenir la coloration syntaxique. Vous pouvez également écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
