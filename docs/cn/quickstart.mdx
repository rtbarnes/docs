## 三步快速上手

在本地运行你的文档站点以预览改动。最佳方式是按顺序完成每一步;在这里打好基础,后续会事半功倍。确保充分理解每一步后再进行下一步。

### 第 1 步:设置本地环境

<AccordionGroup>
  <Accordion title="将文档克隆到本地" icon="copy">
    在引导流程中,如果你之前没有,会为你创建一个包含文档内容的 GitHub 仓库。你可以在[控制台](https://dashboard.mintlify.com)中找到该仓库的链接。

    要将仓库克隆到本地以便进行修改和预览,请参考 GitHub 文档中的[克隆仓库](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)指南。
  </Accordion>
  <Accordion title="启动预览服务器" icon="rectangle-terminal">
    1. 安装 Mintlify CLI:`npm i -g mint`
    2. 进入你的文档目录并运行:`mint dev`
    3. 打开 `http://localhost:3000` 查看实时文档!

    <Tip>
      编辑文件时,预览会自动更新。
    </Tip>
  </Accordion>
</AccordionGroup>

### 第 2 步:部署你的改动

<AccordionGroup>
  <Accordion title="安装我们的 GitHub 应用" icon="github">
    从[控制台](https://dashboard.mintlify.com/settings/organization/github-app)安装 Mintlify GitHub 应用。

    我们的 GitHub 应用会自动将你的改动部署到文档站点,你无需自行管理部署流程。
  </Accordion>
  <Accordion title="更新站点名称与配色" icon="palette">
    作为第一次改动,我们来更新文档站点的名称和配色。

    1. 在编辑器中打开 `docs.json`。
    2. 将 `"name"` 字段改为你的项目名称。
    3. 更新 `"colors"` 以匹配你的品牌。
    4. 保存后,在 `http://localhost:3000` 中可即时看到变化。

    <Tip>
      试着更改主色调,立刻看到效果!
    </Tip>
  </Accordion>
</AccordionGroup>

### 第 3 步:正式上线

<Accordion title="发布你的文档" icon="rocket">
  1. 提交并推送你的改动。
  2. 你的文档会在片刻之间更新并上线!
</Accordion>
